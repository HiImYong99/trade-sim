# 그때 샀더라면

> 비트코인, 테슬라, 삼성전자 등 8개 자산에 과거 특정 날짜에 투자했다면 지금 얼마가 됐을지 계산해주는 시뮬레이터

## 주요 기능

- **8개 자산 지원** — 비트코인, 이더리움, 테슬라, 애플, 삼성전자, KODEX200, 금, S&P500
- **거치식 / 적립식** — 한 번에 전액 투자 또는 매월 정액 투자 시뮬레이션
- **월별 가치 추이 차트** — 투자 기간 동안의 자산 가치 변화 시각화
- **결과 공유** — 토스 공유 기능 연동

## 화면 흐름

```
자산 선택 → 투자 조건 입력 → 광고 → 결과 확인
```

## 기술 스택

| 항목 | 내용 |
|------|------|
| 프레임워크 | React 18 + TypeScript |
| 빌드 도구 | Vite 5 |
| UI | @toss/tds-mobile |
| 플랫폼 | AppsInToss (WebView) |

## 개발 환경 설정

```bash
npm install
npm run dev     # http://localhost:5173
npm run build   # 프로덕션 빌드
npx ait build   # .ait 패키징
```
